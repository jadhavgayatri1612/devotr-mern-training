# VS Code Shortcuts

- Ctrl + P : Open file
- Shift + Alt + Down : Duplicate line
- Ctrl + D : Select same word
- Alt + Up : Move line up
- Shift + Alt + F : Format file
- Ctrl + ` : Open terminal

# Postman Results

# GET All Users
Method: GET
URL: https://jsonplaceholder.typicode.com/users
Status: 200 OK
Response: List of all users

# GET Single User
Method: GET
URL: https://jsonplaceholder.typicode.com/users/1
Status: 200 OK
Response: Single user details

# Create Post
Method: POST
URL: https://jsonplaceholder.typicode.com/posts
Status: 201 Created
Response: New post created

# Update Post
Method: PUT
URL: https://jsonplaceholder.typicode.com/posts/1
Status: 200 OK
Response: Post updated

# Delete Post
Method: DELETE
URL: https://jsonplaceholder.typicode.com/posts/1
Status: 200 OK
Response: Post deleted
